## Drinks API

This app is built with Django Rest Framework.

| Method | url          |
| ------ | ------------ |
| GET    | /drinks/     |
| POST   | /drinks/     |
| GET    | /drinks/{id} |
| PUT    | /drinks/{id} |
| DELETE | /drinks/{id} |
